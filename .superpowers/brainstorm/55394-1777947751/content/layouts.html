Generator layout — pick a direction

Three different ways to organize the workspace. All keep the same functionality — just different flow and feel.

A — Focused form + sidebar

La forma principal ocupa la mayor parte del espacio. La barra lateral a la derecha muestra historial y calidad. Más limpio, menos distracciones. Similar a Notion o Linear.

B — Wizard por pasos

Divide el proceso en 4 pasos: Tema → Coach → Config → Generar. Menos abrumador para usuarios nuevos. Más largo de completar para usuarios avanzados. Similar a Typeform o Onboarding flows.

C — App sidebar permanente (app-like)

Sidebar navy fija a la izquierda con navegación vertical. El contenido principal ocupa el resto. Se siente como una app real — Linear, Vercel, Supabase. Elimina el top navbar.

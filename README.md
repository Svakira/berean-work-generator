# Studio Berean

Generate practical Bible study workbooks and worksheets powered by AI.

## Features

- Student and Teacher modes with different workflow depth
- Study workbooks and student worksheets with full didactic structure
- Reading passage generated by Qwen3 (TotalGPT) — warm, second-person voice
- Biblical citations and academic sources from Berean AI
- Teacher advanced builder: workbook type, timeframe, annotations, improvement focus
- Progress checklist + progress bar per session
- Pre-made workbook library (teens, kids, women, men, husbands, wives)
- Berean transparency panel: source count, collections, model, runtime
- Generation trace panel with prompts and answer previews
- Quality panel (Professor, Student, Interested, Critic) with score and feedback
- Export to Markdown, HTML, and PDF (browser print)
- In-memory cache (12h) to reduce API calls
- Automatic fallback mode when Berean AI is rate-limited
- Light and dark themes

## Setup

1. Clone the repo and install dependencies:

```bash
git clone https://github.com/Svakira/berean-work-generator.git
cd berean-work-generator
npm install
```

2. Copy `.env.example` to `.env` and fill in your API key:

```bash
cp .env.example .env
```

```
TOTALGPT_API_KEY=your_totalgpt_api_key_here
```

Get a TotalGPT API key at [totalgpt.ai](https://totalgpt.ai). The app uses the `Qwen3-30B-A3B` model for reading generation.

> Berean AI (biblical sources) is called automatically — no key needed, but limited to 20 requests/day/IP.

3. Start the server:

```bash
npm start
```

Open: `http://localhost:4173`

## Run with Docker

```bash
docker compose up --build
```

Open: `http://localhost:4173`

## Development

```bash
npm run dev     # same as npm start, watches .env
npm test        # run tests
npm run audit   # minimum quality checks
```

```
  workbookEngine.js  Workbook structure, section builders, markdown output
  bereanClient.js    Berean AI REST client (biblical sources)
  qwenClient.js      TotalGPT/Qwen3 client (reading passage generation)

  index.html         Homepage
  generator.html     Main app shell
  guide.html         Guide viewer
  app.js             Generator logic (do not modify)
  guide.js           Guide viewer logic (do not modify)
  site.js            Site-wide logic (do not modify)
  styles.css         Main stylesheet
scripts/
  audit.js           Quality checks
test/               Basic tests
```

## API Notes

- **Berean AI**: 20 requests/day/IP. When rate-limited, the app falls back to local generation and returns a warning.
- **TotalGPT / Qwen3**: Used for the reading section of study workbooks. Requires `TOTALGPT_API_KEY` in `.env`.
